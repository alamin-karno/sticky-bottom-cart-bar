Adds a sticky Add to Cart and Buy Now button at the bottom of WooCommerce product pages with customizable settings.

== Description ==
A lightweight plugin that lets users quickly add products to cart or buy immediately with a beautiful sticky bottom bar.

== Features ==
* Sticky Add to Cart and Buy Now bar at the bottom of product pages
* Works seamlessly with WooCommerce

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to **Settings > Bottom Cart Bar** to configure display options.

== Frequently Asked Questions ==

= Will this work with any theme? =
Yes, it works with most WooCommerce-compatible themes.

= Can I disable the Buy Now button? =
Yes, go to Settings > Bottom Cart Bar and uncheck "Show Buy Now Button".
